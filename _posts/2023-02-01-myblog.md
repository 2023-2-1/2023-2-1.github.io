# 1. 准备软件

## 1.1 安装 Git

1. 去镜像下载安装包 https://registry.npmmirror.com/binary.html?path=git-for-windows/v2.39.1.windows.1/

2. 配置用户名和邮箱

   ```bash
   git config --global user.name "Redeemer"
   git config --global user.email "1102173427@qq.com"
   ```

   

3. 生成密钥

   ```bash
   ssh-keygen -t rsa -C "1102173427@qq.com" 
   ```

   用`id_rsa.pub`文件中的内容生成**SSH key**

4. ssh

   ```bash
   ssh -T git@github.com
   ```

# 2. 让github托管自己的代码

## 2.1 创建一个github仓库

点击**New**创建远程仓库，作为个人博客使用的远程仓库，仓库名固定格式为：`name.github.io`

![image-20230201165655254](D:/学习笔记/数据库/Redis/图片/image-20230201165655254.png)

## 2.2 创建本地仓库（用于存放博客）



- 需要使用到的 **Git** 命令

```bash
git init											
git add README.md									
git commit -m "first commit"						
git  branch -M main									
git remote add origin git@github.com:2023-2-1/2023-2-1.github.io.git	// 选择 SSH 方式
git push -u origin master						
```

# 3. 部署博客

## 3.1 使用 GitHub Pages 服务部署博客

> Settings ----  Pages

## 3.2 博客主题

1. 下载主题资源

https://github.com/kitian616/jekyll-TeXt-theme

![image-20230201190105493](D:/学习笔记/数据库/Redis/图片/image-20230201190105493.png)



2. 将资源包解压，并将内容copy到**myblog**文件夹中

3. 将新增内容推送到github仓库中

   ```bash
   git add .
   git commit -m "init blog"
   git push
   ```

4. 访问博客 https://2023-2-1.github.io/

5. 个性化配置，在`/myblog/_config.yml`文件中进行配置，然后push到 **github** 仓库
